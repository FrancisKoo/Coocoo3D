# Coocoo3D
一个CPU要求极低的MMD渲染器，支持DirectX12和DXR实时光线追踪。

正在尝试一种新的使用标记语言来扩展渲染过程的方法，只需使用XML写出Pass的描述，渲染器就会按照Pass标记进行渲染，可以让你更灵活的修改渲染流程。

## 基本功能
* 加载pmx模型
* 加载vmd动作
* 播放动画
* 录制图像序列
## 图形功能
* 自定义着色器
* GPU粒子系统
* 烘焙天空盒
* 后处理

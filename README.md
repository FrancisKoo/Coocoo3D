# Coocoo3D
A efficient MMD player.
## 功能
* 播放动画
* 加载pmx模型
* 加载vmd动作
* 加载自定义着色器(.CCShader)
## 特征
* 异步加载物体
* 缓存物体
* 无阻塞UI
## 高效架构
Coocoo3D使用全新的软件开发架构。在这个架构下所有组件都是可以重新加载的，完全消除软件的重启需求，并极大的降低开发难度，提高运行效率。

实现此架构，需要每一个组件都具有Reload方法，并保证能清除之前存在的数据。对于不能一次加载的组件，需要实现多个Reload并每次加载一部分。注意：Reload并不是一个接口，对于具体的实现没有要求。
